1. 정규레슨 환불

취소 시점환불 비율
레슨 3일 전전액 환불 (100%)
레슨 1~2일 전90% 환불 (위약금 10%)
레슨 당일환불 불가

2. 쿠폰레슨 환불

미사용 횟수에 비례하여 환불됩니다.

환불 금액 = 결제 금액 - (사용 횟수 × 1회당 금액)
예) 10회 ₩500,000, 3회 사용 → 환불 ₩350,000

※ 유효기간 만료 후에는 환불이 불가합니다.

3. 코치 책임 취소

코치의 노쇼, 코치 사유로 인한 취소 시 100% 전액 환불됩니다.

4. 환불 불가 안내

  • 레슨 완료(COMPLETED) 후 환불 불가
  • 쿠폰 유효기간 만료 후 환불 불가
  • 정규레슨 당일 수강생 취소 시 환불 불가
